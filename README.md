# PAULO SAT - Landing Page

Site institucional do PAULO SAT - Serviço de instalação e manutenção de antenas parabólicas no Sul Fluminense, RJ.

## 🚀 Tecnologias

- Next.js 14.2.5
- React 18
- CSS Modules

## 📦 Instalação

```bash
npm install
```

## 🔧 Desenvolvimento

```bash
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) no navegador.

## 🏗️ Build para Produção

```bash
npm run build
npm start
```

## 🚢 Deploy na Vercel

Este projeto está otimizado para deploy na Vercel:

1. Conecte seu repositório Git à Vercel
2. A Vercel detectará automaticamente Next.js
3. O deploy será feito automaticamente

### Configurações Importantes

- Node.js: >= 18.17.0
- Build Command: `next build`
- Output Directory: `.next`
- Install Command: `npm install`

## 📁 Estrutura do Projeto

