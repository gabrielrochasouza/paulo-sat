Pill-shaped call-to-action button, the primary conversion element on the site (mostly linking to WhatsApp).

```jsx
<Button variant="primary" href="https://wa.me/...">
  Chamar no WhatsApp
  <ArrowIcon />
</Button>
```

Variants:
- `primary` — yellow fill, dark text, drop shadow. Use once per view for the single main action.
- `secondary` — solid blue fill, white text. Use for a second-tier action.
- `ghost` — translucent white on dark backgrounds (hero, colored panels).
- `outline` — blue border on white, for low-emphasis actions.
